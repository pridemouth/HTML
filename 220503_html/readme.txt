block 속성 / inline 속성
	block level element
		한 줄 통째로 먹음
		p div h1~h6 ul ol form table hr etc.
	inline level element
		오른쪽으로 붙음
		img strong span input textarea label button etc.
	
	display : ___;
	>> block, inline, inline-block, none
	
포지셔닝
	>> position, visibility
	>> z-index(쌓기 순서)
	
	box-sizing: ___;
	>> content-box(내용물 크기를 지정. default), border-box(border 포함 크기를 지정)
	
	float: ___; <!-- 컨텐츠가 inline으로 정렬되는 것처럼 보이게 함. -->
	clear: ___; <!-- 다음 줄에서 float을 사용하고 싶을 때 사용. -->
	>> left, right, none, both
